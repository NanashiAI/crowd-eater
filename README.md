# 🎮 CROWD EATER

敵の群れを喰らいながらボスを倒すブラウザゲームです。  
インストール不要・登録不要、スマホでもPCでもそのまま遊べます。

👉 **[今すぐ遊ぶ](https://nanashiai.github.io/crowd-eater/)**

---

## ゲーム概要

画面を左右に動かして仲間のパワーを増やしながら進み、最後にボスを倒せ！  
ゲートを通過して数を爆増させ、敵を喰らって吸収しながらレベルクリアを目指せ。

---

## 遊び方

1. **START GAME** を押してスタート
2. 画面をタップ／マウスを動かしてキャラを左右に誘導
3. ゲートを通過してパワーを増やす
4. 敵集団に突っ込んで削り合い、全滅させると全パワーを吸収できる
5. ボスが出たら全員で特攻！HPをゼロにすればレベルクリア

---

## ゲームの特徴

| 要素 | 説明 |
|---|---|
| ゲート | `+n`（加算）または `×n`（乗算）でパワーを増やせる |
| 敵集団 | 赤い群れと削り合い。全滅させると初期パワーをまるごと吸収 |
| ボス戦 | 全パワーでボスのHPを削り合う。先にゼロにした方が勝ち |
| 数値表示 | パワーが大きくなると 1.5K・2.5M・1B 形式で表示 |
| レベル進行 | クリアするたびに距離・敵パワー・ボスHPが増加 |

---

## 操作方法

| 操作 | 動作 |
|---|---|
| タップ／スワイプ | キャラを左右に移動（スマホ） |

> ボス戦中は操作不要。全員が自動でボスに群がります。
